# Contributing to Balto Source

Balto Source is a volunteer project built by the community, for the community. Anyone can contribute: you don't have to be a programmer to get involved.

If you want to help the Balto Source project, here's all you need to know.

## For non-technical persons

We are always looking for designer, writers, and artists to improve the look and feel of Balto Source.

You can create artwork, drawings, take screenshots, improve pages or create a new one, or create designs that will end up on the website.

No matter your level of skill, we welcome all contributions.

To know more about what needs contributing, ask away on the Animation Source discord, in the balto-source-2 channel.

### Moderation

This guide only covers contributions to the website itself, it is not about volunteering for moderation roles. So please don't ask to be a moderator here, it is not what this is for. If we need a moderator and you are a right fit, we will get in touch. :)

## For technical persons

Balto Source runs on SvelteKit for the frontend, and Pocketbase + Go for the backend. If you want to contribute to either one, you are more than welcome!

We are mostly in need of frontend devs who want to work on the design and styling part of the website.

The best place to start if you are a first time contributor, is by asking in the balto-source-2 discord channel.

If you want to make a more substantial change, first of all open an issue or comment on an existing issue.
